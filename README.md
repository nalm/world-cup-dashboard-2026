# 2026 FIFA World Cup Interactive Dashboard & Simulator

이 프로젝트는 48개국으로 확장된 **2026 FIFA 월드컵**의 모든 일정과 결과를 한눈에 확인하고 직접 경기 결과를 입력하거나 시뮬레이션할 수 있는 프리미엄 현황판 웹 애플리케이션입니다.

## 주요 기능

1. **대시보드 (Dashboard):**
   - 월드컵 전체 진행 상황 요약 (총 경기 수, 진행률, 골 수, 평균 골 수).
   - "전체 시뮬레이션" 버튼으로 피파 랭킹 기반 가중치로 104개 경기 예측 결과를 일괄 생성 가능.
   - "전체 초기화" 기능으로 입력된 매치 및 토너먼트 결과를 지우고 처음부터 다시 시작 가능.

2. **조별 예선 (Group Stage):**
   - 12개 조(A~L조)의 실시간 순위표 및 72개 경기 스코어 입력 폼.
   - 점수를 입력하는 즉시 각 조의 순위표(승점, 골득실, 다득점 등)가 자동 계산 및 정렬.
   - 각 조의 상위 2개 팀은 자동으로 32강에 진출(초록색 하이라이트).

3. **3위 팀 순위 비교 (3rd Place Comparison):**
   - 12개 조의 3위 팀들의 성적을 실시간으로 비교하여 상위 8개 팀(초록색)과 하위 4개 팀(빨간색)을 자동 계산 및 정렬.

4. **본선 토너먼트 (Knockout Bracket):**
   - 32강전부터 결승전까지의 대진표를 트리 형태로 시각적으로 제공.
   - 조별 예선이 종료되면 FIFA 공식 규정(Annex C)에 기반한 **백트래킹 이분 매칭 알고리즘**을 통해 32강 대진표가 자동으로 채워짐.
   - 토너먼트 스코어를 입력하거나 무작위 예측 결과를 생성하면 즉시 다음 라운드로 승리팀이 진출.

## 기술 스택

- **Frontend:** HTML5, CSS3, ES6 JavaScript
- **Styling:** CSS 변수, Flexbox/Grid 레이아웃, 글래스모피즘 디자인 패턴
- **Icons:** 국가 국기 이모지 활용

## 시작하기

별도의 패키지 설치나 빌드 과정 없이, 브라우저에서 `index.html` 파일을 직접 더블 클릭하여 실행하거나 로컬 웹 서버(Live Server 등)를 통해 열 수 있습니다.
